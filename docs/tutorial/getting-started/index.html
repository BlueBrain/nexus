    Quick Start Guide

    Overview

    This Getting Started tutorial introduces basic Blue Brain Nexus features and capabilities.

    What you’ll build

    • A project within a Blue Brain Nexus deployment where a subset of the MovieLens dataset is ingested and queriable.

    What you’ll learn

    • How to set up the Nexus CLI
    • How to create a project in Nexus
    • How to ingest data
    • How to access data
    • How to share data

    What you’ll need

    • A Python 3.5 or higher with support of Jupyter notebook
